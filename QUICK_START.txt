🖼️ QUICK START - Image Proxy untuk Anime Cover
================================================

❌ MASALAH: Gambar cover anime tidak muncul

✅ SOLUSI: Gunakan Image Proxy

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📍 STEP 1: Jalankan Proxy Server

   Buka Terminal/CMD baru dan jalankan:
   
   cd anime-proxy
   npm install
   npm start
   
   ✓ Tunggu sampai muncul: "Proxy server running at http://localhost:3000"


📍 STEP 2: Jalankan Flutter App

   Di terminal lain:
   
   flutter run
   
   ✓ Gambar anime seharusnya sekarang muncul!

💡 CARA KERJA

Sebelum:  App → Image URL → Server
          ❌ Mungkin diblokir CORS

Sesudah:  App → Proxy URL → Proxy Server → Image URL → Server
          ✅ CORS bypass, caching, stabil

🔧 KONFIGURASI (Jika Diperlukan)

File: lib/config/environment.dart

Untuk EMULATOR ANDROID (ganti localhost dengan 10.0.2.2):
   return 'http://10.0.2.2:3000';

Untuk PRODUCTION (ganti dengan domain proxy Anda):
   static const Environment _env = Environment.production;
   return 'https://proxy.yourdomain.com';

❓ MASALAH UMUM & SOLUSI

Q: Tetap tidak muncul?
A: Pastikan proxy server sudah running di terminal!
   Cek di browser: http://localhost:3000/proxy?target=...

Q: Port 3000 sudah terpakai?
A: Ganti port: PORT=3001 npm start
   Update di environment.dart: return 'http://localhost:3001';

Q: Emulator Android tidak bisa load?
A: Ganti localhost dengan 10.0.2.2
   Di lib/config/environment.dart line 18

Q: Ingin test tanpa proxy?
A: Di environment.dart: static const bool enableImageProxy = false;
